## Features

- webpack 5;
- dev server;
- Babel (polyfills included);
- Handlebars;
- Sass/SCSS;
- PostCSS;
- ESLint, stylelint, Prettier.

## Installation

Clone this repo and npm install.

```bash
npm i
```

## Usage

### Development server

```bash
npm start
```

You can view the development server at `localhost:9000`.

### Production build

```bash
npm run build
```
